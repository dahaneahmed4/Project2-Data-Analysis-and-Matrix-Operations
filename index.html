1.2 Descriptive Statistics Calculation

Below is the rigorous step-by-step calculation for the Mean, Variance, and Standard Deviation for Product C (Column 3) as an example.

Step 1: Calculate the Mean ($\bar{x}$)

Sum of elements divided by count ($n=4$).

$\bar{x} = \frac{12 + 15 + 9 + 9}{4} = \frac{45}{4} = \mathbf{11.25}$

Step 3: Calculate Variance ($s^2$) & Std Dev ($s$)

We use Sample Variance formula ($n-1$) because this is a sample of regions, not the entire population.

Variance: $s^2 = \frac{24.75}{4 - 1} = \frac{24.75}{3} = \mathbf{8.25}$

Detailed Dot Product Calculation (Row 2 Example)

To find the revenue for Region 2 (355), we calculate the dot product of Row 2 and Vector P:

$R_2 = (6 \times 5) + (10 \times 10) + (15 \times 15)$
$R_2 = (30) + (100) + (225)$
$R_2 = \mathbf{355}$

Interpretation: (Prod A Qty $\times$ Price A) + (Prod B Qty $\times$ Price B) + (Prod C Qty $\times$ Price C)

Part 3: Python Implementation (NumPy)

In Data Science, we don't calculate these by hand. We use the NumPy library. Here is the exact code to reproduce these results.

analysis.py 
import numpy as np

# 1. Define the Matrix and Vector
D = np.array([
    [4, 8, 12],
    [6, 10, 15],
    [5, 7, 9],
    [3, 6, 9]
])
P = np.array([5, 10, 15])

# 2. Descriptive Statistics (axis=0 means columns)
means = np.mean(D, axis=0)
std_devs = np.std(D, axis=0, ddof=1) # ddof=1 for Sample Std Dev

print(f"Means: {means}")      # Output: [4.5  7.75 11.25]
print(f"Std Devs: {std_devs}") # Output: [1.29 1.71 2.87]

# 3. Matrix Multiplication (Dot Product)
revenue = np.dot(D, P)

print(f"Revenue Vector: {revenue}")
# Output: [280 355 230 210]

# 4. Identify Max Revenue
max_revenue_index = np.argmax(revenue)
print(f"Best Region: {max_revenue_index + 1}") # Output: 2

Part 4: Conceptual Analysis & Discussion

Interpretation of Variability

We calculated the Coefficient of Variation (CV) ($\frac{\sigma}{\mu}$) for each product.

  • Product A (CV = 0.29): High volatility relative to sales volume.
  • Product B (CV = 0.22): The most consistent performer.

Business Insight: Product B is a reliable "cash cow," while Product A fluctuates more, posing higher inventory risk.

Real-World Matrix Applications

Matrix multiplication is the engine of modern AI.

  • Neural Networks: Weights ($W$) and Inputs ($x$) are multiplied ($W \cdot x$) to calculate neuron activation.
  • Recommendation Systems: Multiplying a User Matrix by an Item Matrix predicts ratings for products users haven't seen yet (Collaborative Filtering).
